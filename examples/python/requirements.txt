# ============================================================
# Version Notes
# ============================================================

# kafka-python 2.0.2:
#   - Latest stable version
#   - Supports Kafka 0.10 - 3.x
#   - Pure Python (no C dependencies)

# confluent-kafka 2.3.0:
#   - Based on librdkafka (C library)
#   - Better performance than kafka-python
#   - Required for Schema Registry / Avro
#   - Requires C compiler for installation

# faust-streaming 0.10.15:
#   - Fork of original faust (no longer maintained)
#   - Active community support
#   - Compatible with Python 3.7+

# ============================================================
# Troubleshooting
# ============================================================

# If confluent-kafka installation fails:
#   - Install build tools:
#     - Ubuntu/Debian: sudo apt-get install build-essential python3-dev
#     - macOS: xcode-select --install
#     - Windows: Install Visual Studio Build Tools

# If faust-streaming installation fails:
#   - Ensure Python 3.7+ is being used
#   - Try: pip install --upgrade pip setuptools wheel
#   - Then: pip install faust-streaming
